# 🚀 JavaScript Básico - Curso em Vídeo

Repositório criado para organizar e documentar todos os exercícios e desafios práticos realizados durante o curso de **JavaScript Básico**, ministrado pelo professor Gustavo Guanabara no [Curso em Vídeo](https://www.cursoemvideo.com/).

## 📌 Sobre o Curso
O curso aborda desde os fundamentos da linguagem até a manipulação do DOM, tratando de temas como:
* Variáveis e Tipos de Dados
* Operadores (Aritméticos, Atribuição, Relacionais e Lógicos)
* Condições (If/Else e Switch)
* Repetições (While, Do While e For)
* Funções e Objetos
* Manipulação de Elementos HTML via JavaScript

## 📂 Estrutura do Repositório
Cada pasta deste repositório corresponde a um módulo ou desafio específico do curso:

* `/exercicios`: Pequenos scripts de fixação de sintaxe.
* `/desafios`: Projetos práticos que integram HTML5, CSS3 e JS (ex: Verificador de Idade, Relógio Dinâmico, Tabuada).

## 🛠️ Tecnologias Utilizadas
* **JavaScript**
* **HTML5**
* **CSS3**
